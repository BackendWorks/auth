## Description

[Nest](https://github.com/nestjs/nest) Microservice framework auth service TypeScript repository.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn dev

# production mode
$ yarn start
```

## Database

```bash
# generate schema
$ yarn generate

# migrate dev
$ yarn migrate

# migrate prod
$ yarn migrate:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
