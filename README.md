## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# production mode
$ npm start

# debug mode
$ npm run debug
```

## Database Migration

```bash
# generate schema
$ npm run generate

# migrate dev
$ npm run migrate

# migrate prod
$ npm run migrate:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
